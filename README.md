# Infinity Bucket

**Infinity Bucket** is a Slimefun 4 (RC-37) addon that enables infinite source generation of water and lava using the Reinforced Water Bucket and Reinforced Lava Bucket.

---

## 📖 Overview

This addon enhances Slimefun gameplay by allowing players to obtain infinite water and lava sources in a controlled and balanced way. It integrates directly with the Reinforced Bucket system, providing a late-game utility for automation, farms, and industrial setups.

---

## ✨ Features

- ♾️ Infinite Water generation using Reinforced Water Bucket  
- 🔥 Infinite Lava generation using Reinforced Lava Bucket  
- ⚙️ Fully integrated with Slimefun 4 (RC-37)  
- 🧩 Lightweight and simple implementation  
- 🛠️ Designed for automation and factory-based gameplay  

---

## 📦 Requirements

- Minecraft Server: **Paper / Spigot**
- Slimefun: **4 (RC-37 or compatible)**
- Java: **17+ recommended**

---

## 🚀 Installation

1. Download the latest release of **Infinity Bucket**
2. Place the `.jar` file into your server’s `plugins` folder
3. Restart or reload your server
4. Ensure Slimefun 4 is installed and running correctly

---

## 🧪 Usage

Once installed, you can use:

- **Reinforced Water Bucket** → Generates infinite water sources
- **Reinforced Lava Bucket** → Generates infinite lava sources

These items behave like standard Slimefun reinforced buckets but with infinite source capability.
